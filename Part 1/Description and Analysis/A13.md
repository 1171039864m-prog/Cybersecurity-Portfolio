I discovered and attempted to use 5 unique online security tools, including: 'EICAR', 'VirusTotal', 'HTTP Observatory', 'Qualys SSL Labs', and 'Norton VPN'.

EICAR is a website that provides test files that contains a special test string known as EICAR test files, which are used to safely test the functionality of antivirus and malware detection systems. These files are harmless but are designed to be detected as threats by security softwares.

'A13 eicar.com' is the content of the EICAR test file 'eicar.com' downloaded from EICAR.


VirusTotal is a online malware analysis platform. Users can upload files or URLs to check for potential threats. the uploaded file is scanned by multiple antivirus engines from different security vendors. Each engine analyses the file independently and reports whether it detects any malicious behaviour. The results will be presented in a report, showing how many engines flagged the file as suspicious or malicious.

'A13 TotalVirus.png' is a screenshot of a section of the report for scanning of an EICAR test file 'eicar.com'.


HTTP Observatory is a tool that analyses a website's HTTP headers and configuration. it checks for the presence and correct implementation of security headers such as Content Security Policy (CSP) , Strict Transport Security (HSTS), and X-Frame-Options.

'A13 HTTP Observatory scan.png' is a screenshot of a section of scan report of 'www.example.com', showing the security headers that it have or have not implemented.


Qualys SSL Labs is a platform that analyses a website's SSL/TLS configuration and gives it a security grade. Users can identify strength of encryption, outdated protocols and misconfigurations that could expose user data.

'A13 Qualys SSL Labs scan.png' is a screenshot showing the report for scanning 'www.uwa.edu.au', showing the website has strong encryption level.


Norton VPN is a VPN service provider. VPN (Virtual Private Network) encrypts internet traffic and routes it through a secure server, protecting user privacy and data from unauthorized access by encrypting user's data and hiding user's IP address. VPN service providers such as Norton VPN provide actual and secured VPN servers worldwide to users. Some provide extra features such as tracking protection.
