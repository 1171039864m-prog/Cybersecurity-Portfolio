I identified a set of Retractable Bollards at Perth CBD. Unlike fixed posts, these can be lowered into the ground to allow authorized vehicles (like emergency services or delivery trucks) to pass.

This is a Dynamic Physical Access Control. It functions similarly to a Firewall Rule: it defaults to a "Deny All" state (up/blocked) and only changes to "Allow" (down/open) when a specific authentication token is provided.

These systems are typically triggered by an encrypted RFID tag, a remote signal, or a central security desk. This highlights a Cyber-Physical dependency—if the network controlling these bollards were hacked, an attacker could remotely "open a hole" in the physical perimeter.

By controlling vehicle access, they protect the physical Availability of the infrastructure behind them, preventing unauthorized vehicles from reaching sensitive areas while maintaining flexible access for legitimate users.
